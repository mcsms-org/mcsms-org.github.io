Project Template Guide

How to use our project template to document and publish your simulation projects.

Getting Started

Our project template (`templates/project-ex.html`) serves as the standard structure for all society projects. It ensures consistency, ease of navigation, and professional documentation for your work.

How to Use

1. Duplicate Template

Copy templates/project-ex.html to your project folder and rename it (e.g., project-name.html).

2. Edit Content

Update the header with your project title and abstract. Fill in the links for your GitHub repo, PDF reports, and release files.

3. Add Contributors

Add a team-card for every member who contributed to the project.

Standard Components

  • Project Meta: Use the status-tag span to indicate if the project is in "Phase 01" or "Released".
  • Core Resources: Always include the repository, reports (Simulation, Technical, Math), and documentation.
  • Contributor Cards: Ensure every contributor has a profile card with their role linked.
